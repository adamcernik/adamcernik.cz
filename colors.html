Naprosto Neznámá Firma - Color Palette

Core Philosophy

The color palette is built around creating a balanced, professional healthcare interface that feels both trustworthy and approachable. The palette establishes a coherent visual language while providing enough flexibility for various UI states and components.

Primary Colors

  • Primary Blue (#B5C5F4) - A soft, calming blue serves as the main brand color, conveying reliability and professionalism without being overwhelming. Use for primary buttons, highlighting important UI elements, and establishing brand recognition.
  • Slate Dark (#394E71) - This deep navy blue provides excellent contrast for text elements while being gentler than pure black. Use for primary text, headings, and important UI elements where strong contrast is needed.
  • Background Main (#FAF7F4) - A warm off-white that creates a softer, more inviting canvas than stark white. This subtle warmth reduces eye strain and creates a more comfortable viewing experience, especially important in healthcare applications where users may spend extended periods using the interface.

Color System Structure

The palette follows a systematic approach with consistent numbering:

  • 50-100: Very light tints (backgrounds, hover states)
  • 200-400: Light to medium tints (subtle UI elements, borders)
  • 500: Base colors (primary elements, default states)
  • 600-900: Darker shades (text, emphasis, active states)

Strategic Accent Colors

  • Accent Yellow - A gentle yellow that provides warmth and can draw attention without anxiety. Use for warnings, highlights, and secondary actions.
  • Accent Salmon - A soft coral tone that creates emotional warmth while providing visual contrast to the blue. Use for highlighting important elements, errors that need attention but aren't critical, and alternative CTAs.

Background System

The background variations allow for creating visual hierarchy through subtle shifts:

  • Background Main - Default canvas
  • Background Warm/Warmer - For creating distinction between UI panels or highlighting content
  • Background Cool/Cooler - For creating emphasis on professional/technical content

Neutrals & System Colors

  • Neutral Scale - A comprehensive grayscale that harmonizes with the primary palette, allowing for subtle differentiation between UI elements without introducing new hues.
  • System Colors - Standardized feedback colors (success, warning, danger, info) maintain expected conventions while being slightly muted to align with the overall aesthetic.

Why These Colors Work Together

  • Harmony & Balance - The palette balances cool tones (blues , slates ) with warm accents (salmon , yellow ) creating a complete visual system that feels neither too clinical nor too casual.
  • Readability & Accessibility - The slate variations provide appropriate text contrast ratios while being less harsh than pure black.
  • Emotional Response - Healthcare applications require a balance of professionalism and empathy. The blue establishes trust and professionalism, while the warm accents add a human touch.
  • Flexibility - The extended palette provides enough range for various UI states (hover, active, disabled, etc.) while maintaining visual coherence.
  • Natural Evolution - The colors are derived from the original brand colors but expanded systematically to provide a complete design system.

Implementation Guidelines

  • Use the slate colors for text hierarchy (darker for headings, lighter for body text)
  • Use the primary blue for interactive elements and brand identity
  • Use accent colors sparingly to highlight important information
  • Use background variations to create subtle visual hierarchy
  • For critical alerts or system messages, use the appropriate system colors

This approach ensures a consistent, professional interface that puts users at ease when dealing with sensitive healthcare information while providing clear visual cues for navigation and interaction.
