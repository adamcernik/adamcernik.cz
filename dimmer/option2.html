Video Dimmer - Option 2

This example shows a video dimmer with a smooth gradient that's darker at the top and bottom, and lighter in the middle area. The dimmer appears when the video is paused and fades out during playback.

Video Title - Smooth Gradient Dimmer Effect
/* Video dimmer with gradient overlay */ .video-dimmer { position: absolute; top: 0; left: 0; width: 100%; height: 100%; background: linear-gradient( to bottom, rgba(0, 0, 0, 0.7) 0%, /* Top gradient start (70% opacity) */ rgba(0, 0, 0, 0.5) 15%, /* Top gradient end (50% opacity) */ rgba(0, 0, 0, 0.5) 85%, /* Bottom gradient start (50% opacity) */ rgba(0, 0, 0, 0.7) 100% /* Bottom gradient end (70% opacity) */ ); opacity: 0; transition: opacity 0.25s ease; } .video-dimmer.paused { opacity: 1; }

How It Works

This dimmer uses a single linear gradient with four color stops to create a smooth transition:

  • Darker (70% opacity) at the top 0%
  • Lighter (50% opacity) at 15% from the top
  • Maintains 50% opacity through the middle section
  • Returns to darker (70% opacity) at the bottom

The dimmer's opacity transitions between 0 (fully transparent) and 1 (fully visible) to show/hide it when the video plays or pauses.
